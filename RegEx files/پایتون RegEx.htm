پایتون RegEx

❮ قبلی بعد ❯

RegEx یا بیان منظم، دنباله ای از کاراکترها است که یک الگوی جستجو را تشکیل می دهد.

از RegEx می توان برای بررسی اینکه آیا یک رشته دارای الگوی جستجوی مشخص شده است یا خیر، استفاده کرد.

ماژول RegEx

پایتون یک بسته داخلی به نام دارد reکه می‌توان از آن برای کار با عبارات منظم استفاده کرد.

وارد کردن reماژول:

import re

RegEx در پایتون

هنگامی که ماژول را وارد کردید re، می توانید شروع به استفاده از عبارات منظم کنید:

مثالسرور پایتون خود را دریافت کنید

رشته را جستجو کنید تا ببینید آیا با "The" شروع می شود و با "Spain" ختم می شود:

import re

txt = "The rain in Spain"
x = re.search("^The.*Spain$", txt)
خودت آن را امتحان کن "

توابع RegEx

ماژول reمجموعه ای از توابع را ارائه می دهد که به ما امکان می دهد یک رشته را برای یک مسابقه جستجو کنیم:

Function Description
findall Returns a list containing all matches
search Returns a Match object if there is a match anywhere in the string
split Returns a list where the string has been split at each match
sub Replaces one or many matches with a string

تابع ()finall

این findall()تابع لیستی را که شامل همه موارد منطبق است برمی گرداند.

مثالسرور پایتون خود را دریافت کنید

چاپ لیستی از همه موارد منطبق:

import re

txt = "The rain in Spain"
x = re.findall("ai", txt)
print(x)
خودت آن را امتحان کن "

فهرست شامل موارد منطبق به ترتیب یافتن آنهاست.

اگر هیچ منطبقی پیدا نشد، یک لیست خالی برگردانده می شود:

مثالسرور پایتون خود را دریافت کنید

اگر مطابقت پیدا نشد، لیست خالی را برگردانید:

import re

txt = "The rain in Spain"
x = re.findall("Portugal", txt)
print(x)
خودت آن را امتحان کن "
 

تابع search()

تابع search()رشته را برای یک تطابق جستجو می کند و در صورت وجود تطابق، شیء Match را برمی گرداند.

اگر بیش از یک مسابقه وجود داشته باشد، فقط اولین مورد بازی برگشت داده می شود:

مثالسرور پایتون خود را دریافت کنید

اولین کاراکتر فضای سفید در رشته را جستجو کنید:

import re

txt = "The rain in Spain"
x = re.search("\s", txt)

print("The first white-space character is located in position:", x.start())
خودت آن را امتحان کن "

اگر هیچ منطبقی یافت نشد، مقدار Noneبرگردانده می شود:

مثالسرور پایتون خود را دریافت کنید

جستجویی انجام دهید که مطابقت نداشته باشد:

import re

txt = "The rain in Spain"
x = re.search("Portugal", txt)
print(x)
خودت آن را امتحان کن "
 

تابع split().

تابع split()لیستی را برمی گرداند که در آن رشته در هر مسابقه تقسیم شده است:

مثالسرور پایتون خود را دریافت کنید

در هر نویسه فضای سفید تقسیم کنید:

import re

txt = "The rain in Spain"
x = re.split("\s", txt)
print(x)
خودت آن را امتحان کن "

با تعیین پارامتر می توانید تعداد رخدادها را کنترل کنید maxsplit :

مثالسرور پایتون خود را دریافت کنید

رشته را فقط در اولین اتفاق تقسیم کنید:

import re

txt = "The rain in Spain"
x = re.split("\s", txt, 1)
print(x)
خودت آن را امتحان کن "
 

تابع () sub

تابع sub()مطابقت ها را با متن انتخابی شما جایگزین می کند:

مثالسرور پایتون خود را دریافت کنید

هر کاراکتر فضای سفید را با عدد 9 جایگزین کنید:

import re

txt = "The rain in Spain"
x = re.sub("\s", "9", txt)
print(x)
خودت آن را امتحان کن "

با تعیین پارامتر می توانید تعداد جایگزین ها را کنترل کنید count :

مثالسرور پایتون خود را دریافت کنید

2 مورد اول را جایگزین کنید:

import re

txt = "The rain in Spain"
x = re.sub("\s", "9", txt, 2)
print(x)
خودت آن را امتحان کن "
 

مطابقت شی

Match Object یک شی است که حاوی اطلاعاتی در مورد جستجو و نتیجه است.

توجه: اگر مطابقت وجود نداشته باشد، مقدار Noneبه جای Match Object برگردانده می شود.

مثالسرور پایتون خود را دریافت کنید

جستجویی انجام دهید که یک Match Object برگرداند:

import re

txt = "The rain in Spain"
x = re.search("ai", txt)
print(x) #this will print an object
خودت آن را امتحان کن "

شی Match دارای ویژگی ها و روش هایی است که برای بازیابی اطلاعات مربوط به جستجو و نتیجه استفاده می شود:

.span()یک تاپل حاوی موقعیت‌های شروع و پایان مسابقه را برمی‌گرداند.
.stringرشته ارسال شده به تابع را برمی گرداند
.group()بخشی از رشته را که در آن تطابق وجود داشته است

مثالسرور پایتون خود را دریافت کنید

موقعیت (موقعیت شروع و پایان) اولین وقوع مسابقه را چاپ کنید.

عبارت منظم به دنبال هر کلمه ای است که با حرف بزرگ "S" شروع می شود:

import re

txt = "The rain in Spain"
x = re.search(r"\bS\w+", txt)
print(x.span())
خودت آن را امتحان کن "

مثالسرور پایتون خود را دریافت کنید

رشته ارسال شده به تابع را چاپ کنید:

import re

txt = "The rain in Spain"
x = re.search(r"\bS\w+", txt)
print(x.string)
خودت آن را امتحان کن "

مثالسرور پایتون خود را دریافت کنید

قسمتی از رشته را که در آن مسابقه وجود دارد چاپ کنید.

عبارت منظم به دنبال هر کلمه ای است که با حرف بزرگ "S" شروع می شود:

import re

txt = "The rain in Spain"
x = re.search(r"\bS\w+", txt)
print(x.group())
خودت آن را امتحان کن "

توجه: اگر مطابقت وجود نداشته باشد، مقدار Noneبه جای Match Object برگردانده می شود.
